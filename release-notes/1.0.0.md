# UniGeek Firmware v1.0.0
*2026-03-08*

First public release — multi-device firmware built from the ground up to support three ESP32 boards with a unified codebase.

## Supported Boards

> **M5StickC Plus 1.1**
> Button navigation, AXP192 power management.

> **LilyGO T-Lora Pager**
> Full keyboard, rotary encoder, BQ25896 power.

> **M5Stack Cardputer**
> Full keyboard, SD card, compact form factor.

## Features

> **WiFi Toolkit**
> Packet monitor, deauth detector, EAPOL capture with offline WPA2 brute-force, beacon spam, and WiFi analyzer.

> **BLE Scanner**
> Scan and list nearby BLE devices.

> **Network Tools**
> Port scanner, IP scanner, IP ping, and web-based file manager accessible from any browser on the same network.

> **BadUSB / HID**
> USB keyboard emulator with DuckyScript support for automated payloads.

> **ESP-NOW Chat**
> Peer-to-peer wireless chat between two UniGeek devices, no WiFi network required.

> **File Manager**
> Browse SD card and LittleFS files on device.

> **QR Code Generator**
> Render QR codes on display from text or URLs.

> **I2C Detector**
> Scan I2C bus and display found device addresses.

> **RTC / Clock**
> Real-time clock display with NTP sync support.

> **AI Agent**
> On-device conversational AI assistant via WiFi.

> **Games**
> Wordle and HEX Decoder.

## Install

Flash the `.bin` file for your device using ESP Web Flasher or `esptool.py`. Copy the `sdcard/` folder to your SD card root for scripts, portals, and wordlists.
