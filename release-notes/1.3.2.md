# UniGeek Firmware v1.3.2

## New Features

> **Flappy Bird Game**
> Classic side-scroller game with randomized pipes and scoring

> **Configurable EAPOL Capture**
> Channel hopping support and configurable capture settings

> **File Manager Sorting**
> Files and directories now sorted for easier navigation

## Bug Fixes

- Fix screen-off stopping background processes (EAPOL, GPS, Karma, etc.)
- Fix Cardputer sound output
- Fix EAPOL password counter for built-in wordlist
- Fix SSID name capture
- Fix display flickering on multiple screens
- Fix power management rules (PowerSave vs PowerOff)
- Fix LogView status bar positioning
- Fix large file handling in file viewer
- Fix Web File Manager power-off issue

## Supported Boards

- M5StickC Plus 1.1
- M5StickC Plus 2
- M5Stack Cardputer
- M5Stack Cardputer ADV
- LilyGO T-Lora Pager
- LilyGO T-Display 16MB
