# UniGeek 1.6.0

## New Boards

> **M5Stack CoreS3** (M5Unified) — touch navigation, I2S speaker, full USB HID support
> **M5StickC S3** — I2S speaker, USB HID, IR transmit
> **LilyGO T-Display S3** — USB HID
> **LilyGO T-Embed CC1101** — rotary encoder, I2S speaker, CC1101 module
> **DIY Smoochie** — 5-button nav, CC1101 module
> **CYD (ESP32-2432S028R)** — budget-friendly board support

Six new boards in one release. Total supported: 12 boards.

## New Features

> **NRF24L01+ Module**
> 2.4 GHz spectrum analyzer, multi-mode broadcaster (10 presets + single-channel + hopper), and wireless HID receiver scanner — requires NRF24L01+ module

> **Chameleon Ultra BLE Client**
> Full BLE client for ChameleonUltra and ChameleonLite: device info, 8-slot card manager, HF/LF scan and clone, MIFARE dictionary attack + dump + magic card detection, MFKey32 log recovery, T5577 write, password cleaner

> **Hand Orientation Toggle**
> Flip display and swap UP/DOWN navigation — M5StickC Plus 1.1, Plus 2, and StickC S3

> **Speaker Test Screen**
> Play Win, Lose, Notification, and Beep sounds to verify speaker output

> **Touch Navigation Overlay**
> Live edge indicators for touch areas — always visible during navigation (CoreS3, CYD)

## Improvements

- **Sprite rendering** — all screens migrated to per-region sprites; significantly lower peak RAM usage across all boards
- **Ducky Script** — expanded command support with more script commands recognized
- **GPS screen** — default pin auto-configuration per board type
- **Navigation** — back navigation now consistent on all boards; screen no longer wakes on nav press
- **Achievement system** — reworked tracking, expanded achievement entries

## Bug Fixes

- Input popups (text, number, select): back/cancel no longer saves changes
- Web File Manager: storage file list and upload path corrections

## Install

Browser installer (no tools required): https://unigeek.xid.run
Download firmware binaries: https://github.com/lshaf/unigeek/releases

Copy `sdcard/` folder to SD card root, or use the built-in Download menu on the device.
