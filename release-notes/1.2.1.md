# UniGeek Firmware v1.2.1
*2026-03-23*

Maintenance release — hardware fix for T-Display 16MB battery readings.

## Bug Fixes

- **T-Display 16MB battery** — fixed an incorrect battery ADC pin reference that could cause wrong battery readings

## Install

Download binaries from the [GitHub Releases page](https://github.com/lshaf/unigeek/releases). Copy the `sdcard/` folder to your SD card root, or use the built-in **Download** menu in the firmware.
