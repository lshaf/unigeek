# UniGeek 1.8.2
*2026-05-26*

A big sub-GHz-and-connectivity cycle: a new M5 RF433 module, two new ways to manage device files (over BLE and over USB serial in the browser), HID media/camera controls, and a new T-Display-S3 Touch board — plus KeeLoq rolling-code decode/replay, DuckyScript 3.0, and a much richer Lua scripting environment.

## New Boards

> **T-Display-S3 Touch**
> New env for the LilyGO T-Display-S3 Touch — hybrid hardware buttons plus a CST820 capacitive touchscreen.

## New Features

> **M5 RF433 Module**
> Capture, replay, and jam 433.92 MHz signals using the M5 RF433T/R two-pin Grove units — no CC1101 needed. Shares the .sub file format and capture UI with Sub-GHz.

> **BLE File Manager**
> Manage on-device files wirelessly over BLE (Nordic UART) — no WiFi required. Connect from the browser at https://unigeek.xid.run/app/files; includes a save shortcut and flow control.

> **USB Serial File Manager**
> Browse and edit device files straight from the browser over USB serial at https://unigeek.xid.run/app/files — view/edit text, hex view, image preview, and drag-and-drop upload. Can be toggled off in Settings to free ~12 KB SRAM.

> **HID Media / Camera Controls**
> New submenu of consumer-control keys — volume, playback, and camera shutter — over both USB and BLE.

## Improvements

- **Sub-GHz (CC1101)** — receive filter (RAW / Code, now defaulting to Code), scrollable Signal Info view, and a unified Send tap popup showing all options
- **KeeLoq rolling-code** — auto-decode on capture plus counter-step "Replay +1" replay; Mfcodes keystore menu with a default keystore shipped at `/unigeek/mfcodes` (Sub-GHz and M5 RF433)
- **Lua Runner** — new `uni.wifi` + `uni.http` (blocking GET/POST, TLS, auto-cleanup on exit) and `uni.subghz` (auto-detect CC1101 / M5 RF433) bindings; VM heap and script source now run from PSRAM on PSRAM boards; board-conditional task stack (32 KB with PSRAM, 8 KB without); on-device Lua Scripts download browser; hardware-entropy `math.random` seeding
- **DuckyScript 3.0** — VAR / IF / WHILE / FUNCTION and expression subset, with a demo script
- **EAPOL capture** — new Target mode that deauths only the chosen BSSID after a 10 s WiFi scan
- **BLE spam** — fixed the iOS popup and boosted TX power / interval for Android and Samsung
- **File navigation** — faster directory listing, a ".." parent entry across modules, and pickers confined to their subtree
- **Password Manager** — WebAuthn-derived entries via HMAC(master.bin, pw|fields); Source row hidden on non-WebAuthn boards
- **Install page** — install any past version, with a per-version board filter
- **On-screen keyboard** — added `_ \ |` symbols
- **StickC Plus** — bit-bang the encoder HAT I²C bus so it stops fighting Wire/ExI2C
- **Website** — Discord link in the footer; privacy-respecting install analytics

## Bug Fixes

- **Speaker** — fix `tone()` going silent after rapid bursts (timing race + idle starvation)
- **Status overlay** — restore the caller's text datum so it no longer leaks MC_DATUM
- **File browser** — pass the directory by value so the ".." row can't clobber the selected path
- **Board metadata** — deep-dive tag rewrite to match each board's pins_arduino.h reality

## Install

Install in your browser at https://unigeek.xid.run — no tools required.
Or download a `.bin` from https://github.com/lshaf/unigeek/releases and flash with esptool / PlatformIO.

Copy the `sdcard/` folder to the SD card root, or use the built-in **Download** menu on the device to fetch sample files over WiFi.
