# BLE Detector

Passively scans for nearby Bluetooth Low Energy devices and identifies potential threats. Accessed from **Bluetooth > BLE Detector**.

## What It Detects

| Type | Description |
|------|-------------|
| **Flipper Zero** | Detects real Flipper Zero devices and distinguishes them from spoofed ones |
| **Credit Card Skimmers** | Identifies Bluetooth modules commonly used in skimming hardware |
| **Apple AirTags / FindMy Trackers** | Detects Apple tracking devices; shows estimated distance and movement trend (approaching/stationary/moving away) |
| **BitChat App** | Alerts when someone nearby is running the BitChat mesh messaging app |
| **BLE Spam Attacks** | Detects spam patterns targeting Apple, Android, Samsung, and Windows devices |

## Usage

1. Go to **Bluetooth > BLE Detector**
2. The scanner runs continuously, listing detected threats in real-time
3. Each entry shows the device type, name (if available), signal strength, and MAC address
4. Press **BACK** or **Press** to stop scanning
