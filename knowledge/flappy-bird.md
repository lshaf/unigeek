# Flappy Bird

A classic side-scrolling arcade game. Guide a yellow bird through gaps in green pipes without hitting them, the ground, or the ceiling.

---

## How to Play

The bird falls continuously under gravity. Press or tap to flap and gain upward momentum. Pipes spawn from the right with randomized gap positions. You score one point each time the bird passes through a pipe gap. The game ends on any collision — with a pipe, the ground, or the ceiling.

Pipe gap size and speed scale with the screen dimensions, so the game adapts to each device's display size automatically.

---

## Controls

### Keyboard Boards
*(T-Lora Pager, M5 Cardputer, M5 Cardputer ADV)*

| Action | Key |
|--------|-----|
| Flap | `Space` or `Enter` |
| Exit to menu | BACK |

### Touch Boards
*(M5 CoreS3 Unified)*

| Action | Gesture |
|--------|---------|
| Flap | Tap the screen |
| Hold to flap repeatedly | Hold the screen |
| Exit to menu | Swipe BACK edge |

### Button / Encoder Boards
*(M5StickC Plus 1.1, M5StickC Plus 2, T-Display, T-Display S3, T-Embed CC1101, M5Stick S3, DIY Smoochie, CYD variants)*

| Action | Input |
|--------|-------|
| Flap | SELECT or UP |
| Exit to menu | BACK |

## High Scores

The top 5 scores are saved to `/unigeek/games/flappy_hs.txt`. There is no difficulty setting — a single global leaderboard is shared across all sessions. The first-place score is highlighted in yellow.

The menu screen shows your last score and personal best. The result screen shows a "NEW HIGH!" banner if your run entered the top 5.
